틸만 인덕션의 특징과 주요기능

인덕션이란?

자기장으로 인해 발생하는 유도전류를 열로 바꿔서 조리하는 가열기입니다 인덕션 방식은 전자유도 가열방식으로 자성에 반응하는 조리기구(철 스테인레스 등)를 인덕션 내부에 자력선이조리기구 속의 철 성분과 반응하여,냄비 자체만을 뜨겁게 만들어 음식을 조리하는 방식을 말합니다

잠금기능으로 사고예방
아이들의 잘못된 사용으로 인한안전사고를 방지하도록 전원 이외의 버튼이 모두 잠기는 기능

슬라이드 터치 방식으로 쉽게
슬라이드 터치 방식으로 원하는 온도를 쉽게 조절할 수 있는 기능

보온기능으로 오래오래
음식이 식을 걱정은 NO! 적당한 온도를 유지하게 만들어 음식을 오래도록 따뜻하게 하는 기능

파워 부스터로 더 강하게
빠른 조리가 필요할 때 가장 높은 온도를 설정할 수 있도록 하는 기능

타이머 기능으로 편리하게
자리를 비워도 다른 일을 동시에 하고 있어도 걱정은 STOP! 원하는 시간에 작동이 멈추는 기능

틸만 인덕션의 장점

  • • 조리도구에 에너지를 직접 전달하므로 에너지를 절약합니다.
  • • 제품 위에 조리기구가 올려져 있을 때만 열이 발생함으로 더욱 안전합니다
  • • 상판이 아닌 조리기구 바닥을 통해서만 열이 가해지기 때문에 음식물이 넘쳐도 상판에 쉽게 눌러붙지 않습니다
  • • 열 공급이 빠르고 세밀한 온도 조절이 가능합니다
  • • 매우 빠른 조리 방식으로 조리 시간이 짧아져 시간이 절약됩니다
  • • 조리구가 직접 달궈지지 않아 청소와 관리가 간편합니다
